Solicitud de Simulación de Crédito

Este programa le permitirá hacer una simulación crediticia, para crear un plan de pagos para un préstamo del nuevo fondo de crédito MiVivienda. Los pagos se calcularán con el método Francés (Como Michelle) Vencido Ordinario, podrán ser calculados tanto en soles como en dólares, y se podrá elegir el plazo en el que se planea devolver el crédito. Este simulador también permite elegir el tipo de tasa (Efectiva o Nominal) que se usará en el cálculo, así como el interés de la misma. Por último, también le permitirá establecer un periodo de gracia inicial parcial o total
